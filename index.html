Horaires des étudiants et étudiantes par groupe ainsi que l'horaire des profs pour le quadrimestre en cours.

Tip : il est possible de copier le lien d'un horaire dans son agenda. Ceci, quel que soit l'outil d'agenda utilisé (poési, owncloud, gcalendar…) qui supporte le format ics. Le lien se trouve en bas de page. Si vous souscrivez à l'agenda (plutôt que d'importer les évènements), celui-ci sera mis à jour automatiquement.

Code jaune (bac 1) / orange (bac 2, 3) : si vous avez cours à distance et un cours à l'école dans la même demi-journée, vous pouvez demander un local pour suivre le cours à distance auprès de la direction.
